About the Receiver

The SMART C-CDA Receiver provides a lightweight Web API to clinical summary data. It exposes discrete JSON resources for:

  • C-CDA Documents
  • C-CDA Sections
  • C-CDA Organizers
  • C-CDA Entries

To learn more, visit the Documentwation wiki on GitHub.
